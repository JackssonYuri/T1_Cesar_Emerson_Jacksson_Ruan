Instruções de compilação :
Primeiro compilar o makefile com o comando:

make all.

Após isso vão ser gerados os arquivos com extensão .o, dessa forma agora pode-se compilar o main com o comando:

./main

//Caso já possua os arquivos com extensão .o, desejável que realize o comando:

make rm

#include <stdio.h>
#include <ncurses.h>
#include <unistd.h>
#include <string.h>
